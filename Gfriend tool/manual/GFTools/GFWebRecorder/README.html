GFriend Web Recorder

GFriend Web Recorder support to making GFriend TC using by Web keywords. When you use this tool, this tool records your actions from Chrome browser. Tester can export this recording data to GFriend TC.

1. Install

  1. If you didn't install this Chrome extension on your local PC, you can install it by clicking "Tool > External Tools > GF Web Recorder".
  2. After installation, GF Web Recorder icon is displayed to the Chrome browser.

NOTE: If you can not find GF Web Recorder icon after clicking the option, retry it after killing all Chrome process.

2. Execute

You can start GFWebRecorder by clicking button on the chrome browser.

3. Record

  1. You can find text box for Web browser after clicking Record button.
  2. If you type "url" that you want to use it, it makes new tabs for recording.
  3. It will be recorded on the tools what you act on the browser.

NOTE: You can stop the recording by clicking Stop button. If you want to restart recording, click Record button again. It will start recording again without creating new tab.

4. Record result

  1. You can find your running script form the main UI. It usually use xPath for testing.

5. Export

  1. It display GFriend TC that makes automatically when you click Export button.
  2. You can select two type of formats on the UI: GFriend, and XML
    • If you want to copy it to Clipboard, click "Copy to Clipboard" button.
    • If you want to Save it to specific file, click "Save As File..." button.

[More information]

  1. Some types of pop-up are not recorded to this tool. If you want to make TCs include these types pop-up, you should add "id" and "password" to your Web.Open() keyword. At this case, you should add "{id}:{password}@" after "https://".

    Web.Open with Chrome(https://{url}) -> Web.Open with Chrome(https://{id}:{password}@{url})

    ex) Web.Open with Chrome(https://admin:admin@130.31.193.111:3443/mymfp/conf)
